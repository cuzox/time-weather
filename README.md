# Time Weather
Return the time and weather for given comma separated locations

# Test
```npm run test```

# Run
The input can be comma separated args or strings

```npm run weather new york, new delhi, "paris, london"```

## Output
```
There's clear sky in New York (US), it's 5:55 PM and the temperature is 63.7 째F
There's haze in New Delhi (IN), it's 3:25 AM and the temperature is 80.6 째F
There's scattered clouds in Paris (FR), it's 11:55 PM and the temperature is 51.4 째F
There's scattered clouds in London (GB), it's 10:55 PM and the temperature is 45.72 째F
```
